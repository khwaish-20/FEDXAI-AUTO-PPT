FEDXAI-AUTO

"A Federated, Privacy-Preserving AI Framework for Predictive Maintenance in Ethanol-Blended (E20) Vehicles."

Problem Statement

The automotive industry is shifting towards proactive Predictive Maintenance (PdM), particularly to handle new challenges like accelerated part degradation caused by E20 fuel. However, deploying AI for this purpose is currently blocked by two massive hurdles: (1) The need to centralize private driver data to train models, which violates user privacy, and (2) The 'Black Box' nature of AI, which gives drivers opaque warnings without explaining why a part is failing, destroying user trust.

Literature Review / Market Research

Current PdM systems face two major roadblocks. First, traditional methods rely on centralized data sharing, which violates data confidentiality and security regulations. Second, these AI models suffer from a lack of interpretability; they act as opaque "black boxes," making it difficult for operators to trust their failure predictions. Recent 2025 research successfully applied Explainable Federated Learning (XFL) to static factory machines, FedXAI-Auto bridges the gap to the dynamic automotive sector.

Research Gap / Innovation

A recent 2025 study successfully developed a privacy-preserving Explainable FL (XFL) model for sustainable manufacturing, achieving an impressive 98.15% accuracy in a factory setting. In FEDXAI-AUTO, we innovate by deploying a lightweight, decentralized Temporal Model (CNN+LSTM) onto an ESP32 edge device, ensuring zero raw-data leakage while using XAI to translate complex E20 degradation signatures into human-readable alerts for drivers.

System Methodology

Dataset / Input

Digital Twin Data Factory: Generated 10,000 unique vehicle silos using physics-based simulations of E20 fuel degradation curves (Pump/Filter wear).

Model / Architecture

Hybrid 1D-CNN Architecture: optimized for Time-Series sensor data. Uses Weighted Loss to detect rare failures and SHAP for explainability.

Live Edge Execution

Running on ESP32 Edge Hardware (TinyML).
AVG Inference Time: 12ms.

Results & Analysis

Accuracy / Performance 98.64%

Achieved 98.64% Accuracy on unseen vehicles (Honest Split). Our model reduced False Negatives by 40% compared to the baseline, ensuring critical failures are never missed. The model size was compressed to 22KB for edge deployment.
